# Intro

the name of this site, nokit, or No Kit,
comes from the idea that learning
 is best done by figuring things out. 
That is, instead of following inatructions 
and a premade plan, start with a general idea 
of what the outcome might look like and work
 towards that. it will take longer of course, 
but the journey will hopefully be both more 
interesting and the result more satisfactory.
